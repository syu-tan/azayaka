Azayaka documentation
=====================

.. image:: ../../figure/001_project_icon.png
   :alt: Azayaka project icon
   :align: center
   :width: 240

`azayaka` is a Python package for SAR (Synthetic Aperture Radar) processing and a simple InSAR Project.

and QGIS plugin SAR tools. It focuses on geometry calculations for JAXA CEOS and SICD formats.
